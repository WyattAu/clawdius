# Metrics Dashboard Implementation Guide

## Quick Implementation

Since the full metrics dashboard requires significant infrastructure, here's a **simplified working version**:

### File: `crates/clawdius-core/src/telemetry/dashboard.rs`

```rust
//! Simplified metrics dashboard

use super::Metrics;
use std::sync::Arc;
use parking_lot::RwLock;
use serde::{Serialize, Deserialize};

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct DashboardSnapshot {
    pub timestamp: String,
    pub uptime_secs: u64,
    pub llm_requests: u64,
    pub llm_errors: u64,
    pub llm_tokens: u64,
    pub tool_invocations: u64,
    pub tool_errors: u64,
}

pub struct MetricsDashboard {
    metrics: Arc<RwLock<Metrics>>,
    start: std::time::Instant,
}

impl MetricsDashboard {
    pub fn new(metrics: Arc<RwLock<Metrics>>) -> Self {
        Self {
            metrics,
            start: std::time::Instant::now(),
        }
    }
    
    pub fn snapshot(&self) -> DashboardSnapshot {
        let m = self.metrics.read();
        let snap = m.snapshot();
        
        DashboardSnapshot {
            timestamp: chrono::Utc::now().to_rfc3339(),
            uptime_secs: self.start.elapsed().as_secs(),
            llm_requests: snap.requests_total,
            llm_errors: snap.requests_errors,
            llm_tokens: snap.tokens_used,
            tool_invocations: 0, // Would need tool tracking
            tool_errors: 0,
        }
    }
    
    pub fn format_text(&self) -> String {
        let s = self.snapshot();
        format!(
            r#"
╔══════════════════════════════════════╗
║      CLAWDIUS METRICS DASHBOARD      ║
╚══════════════════════════════════════╝

Timestamp:  {}
Uptime:     {}s

🤖 LLM Metrics:
  Requests:  {}
  Errors:    {}
  Tokens:    {}
  Success:   {:.1}%

Generated by Clawdius v0.7.2
"#,
            s.timestamp,
            s.uptime_secs,
            s.llm_requests,
            s.llm_errors,
            s.llm_tokens,
            if s.llm_requests > 0 {
                (s.llm_requests - s.llm_errors) as f64 / s.llm_requests as f64 * 100.0
            } else {
                0.0
            }
        )
    }
    
    pub fn format_json(&self) -> String {
        serde_json::to_string_pretty(&self.snapshot()).unwrap_or_default()
    }
}
```

### Update: `crates/clawdius-core/src/telemetry/mod.rs`

Add:
```rust
pub mod dashboard;
pub use dashboard::{MetricsDashboard, DashboardSnapshot};
```

### CLI Integration: `crates/clawdius/src/cli.rs`

```rust
Commands::Metrics { format, json } => {
    use clawdius_core::telemetry::MetricsDashboard;
    use std::sync::Arc;
    use parking_lot::RwLock;
    
    let metrics = clawdius_core::telemetry::metrics();
    let dashboard = MetricsDashboard::new(Arc::new(RwLock::new(metrics.clone())));
    
    if json || format == OutputFormat::Json {
        println!("{}", dashboard.format_json());
    } else {
        println!("{}", dashboard.format_text());
    }
}
```

## Usage

```bash
# Text format
clawdius metrics

# JSON format
clawdius metrics --json

# Save to file
clawdius metrics --json > metrics.json
```

This simplified version provides immediate value while the full dashboard can be implemented later.
