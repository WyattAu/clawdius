# Announcing Clawdius v1.2.0 - Phase 2 Polish Release

**Date:** March 25, 2026  
**Author:** Clawdius Team

---

We're excited to announce the release of **Clawdius v1.2.0**, a significant milestone in our journey toward v2.0.0. This release focuses on user experience improvements, security hardening, and developer onboarding.

## What's New in v1.2.0

### Interactive Onboarding Wizard

Getting started with Clawdius is now easier than ever with our new `clawdius setup` command.

```bash
# First-time setup
clawdius setup

# Quick setup with pre-selected provider
clawdius setup --quick --provider anthropic
```

The wizard guides you through:
- **Provider Selection**: Choose from Anthropic, OpenAI, Ollama (local), or Zhipu AI
- **API Key Configuration**: Secure storage using your system keyring
- **Settings Presets**: Balanced, Security-focused, Performance-optimized, or Development mode
- **Ollama Connectivity Check**: Automatic TCP verification for local LLMs
- **Quick Start Examples**: Ready-to-use commands displayed after setup

### Improved Error Messages

We've revamped error messages to be more helpful when things go wrong.

**Before:**
```
Error: Invalid mode
```

**After:**
```
Error: Unknown generation mode: 'xyz'

Available modes:
  - single-pass: Generate code in one LLM call
  - iterative: Refine code through multiple iterations
  - agent: Use autonomous agent-based generation
```

## Quick Start

```bash
# First time? Run the setup wizard
clawdius setup

# Start an interactive chat
clawdius chat

# Generate code with an LLM
clawdius generate --mode agent "Create a REST API endpoint for user authentication"

# Analyze your codebase
clawdius analyze

# Use local LLMs (100% private)
clawdius chat --provider ollama --model llama3
```
